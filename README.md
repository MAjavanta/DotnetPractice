# Dotnet Practice

A series of projects done to teach myself .NET incrementally, focusing on SOLID principles and best practices.

1. Calculator - One file project learning the basic syntax and structure of Program.cs
2. ToDo App - Learning methods, loops, switch statements, validating user input.
3. Dice Rolle Game - Learning OOP, creating classes using the Single Responsibility Principle, using switch expressions, clean architecture, Dependency Injection
4. Cookies Cookbook - Implementing interfaces, sharp focus on clean code and well designed architecture, particularly looking at designing good interface for classes, reading and writing to file system and persisting user choices
